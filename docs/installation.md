# Installation

## Stable release

To install Care Quick Assign, run this command in your terminal:

```sh
pip install Care Quick Assign
```

## From source

The source files for Care Quick Assign can be downloaded from the [Github repo](https://github.com/ohcnetwork/care_quick_assign).

You can either clone the public repository:

```sh
git clone git://github.com/ohcnetwork/care_quick_assign
```

Or download the [tarball](https://github.com/ohcnetwork/care_quick_assign/tarball/master):

```sh
curl -OJL https://github.com/ohcnetwork/care_quick_assign/tarball/master
```

Once you have a copy of the source, you can install it with:

```sh
cd care_quick_assign
pip install .
```
